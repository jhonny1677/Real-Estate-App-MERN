📝 Manual Testing Steps

  1. Start the servers: 
    # Terminal 1 - API Server
cd api && npm start

# Terminal 2 - Client
cd client && npm run dev

# Terminal 3 - Socket Server  
cd socket && node app.js
  2. Update .env file with your email credentials
  3. Test Registration: Go to /register and create an account
  4. Check Email: Look for verification email in your inbox
  5. Test Login: Try logging in before/after verification
  6. Test Password Reset: Go to /forgot-password
  7. Test Image Loading: Go to /list and check network tab
  8. Test Advanced Filters: Use "Show Advanced Filters" on /list page
